JSON Object Example

📘 What is JSON?

JSON (JavaScript Object Notation) is used to store and transfer data in a structured format. It is widely used in APIs and web applications.

🧠 Dot Notation

Dot notation is used to access object properties.

Example: student.name returns the name of the student.

📌 Key Features

  • Lightweight data format
  • Easy to read and write
  • Used in web services and APIs
  • Supports key-value pairs
