AngularJS Select Elements: ng-options vs ng-repeat

1. Using ng-options (Recommended for select elements)

ng-options automatically creates <option> elements and is more efficient:


Selected: {{selectedPerson1 ? selectedPerson1.name + ' (Age: ' + selectedPerson1.age + ')' : 'None'}}

2. Using ng-repeat (Creates option elements manually)

ng-repeat creates <option> elements explicitly:


Selected: {{selectedPerson2 ? selectedPerson2.name + ' (Age: ' + selectedPerson2.age + ')' : 'None'}}
